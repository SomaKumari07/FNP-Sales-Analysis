🌸 Ferns and Petals (FNP) Sales Analysis Dashboard 📊
📌 Project Overview

This project is an interactive Sales Analysis Dashboard created for Ferns and Petals (FNP) to analyze sales performance, customer behavior, delivery trends, and product performance across different occasions like 🎉 Diwali, 💖 Valentine’s Day, 🎂 Birthdays, 🎁 Anniversaries, and more.

The dashboard helps in generating business insights and supports data-driven decision-making.

🎯 Objectives
Analyze overall sales and revenue performance 💰
Track monthly sales trends 📈
Identify top-performing products 🏆
Understand customer spending behavior 👥
Analyze delivery performance 🚚
Compare revenue across occasions 🎊
Identify top cities by number of orders 🌍
